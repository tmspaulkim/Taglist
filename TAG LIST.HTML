    가장 처음 등장하는 문서의 종류를 언급하는 표시 ===> !doctype html

    각 코딩 시작에 meta charset="utf-8" 이라는 명령을 해줌으로써, 한글 인식이 가능

    고위직 태그들: "Html", "Head","Body"

    특별한 기능은 없지만, 전체 코딩의 시작 부분과 본문을 구분하여 주는 역할을 함으로 반드시 사용하여 코딩의 전체적인 구조의 윤곽을 나타낼 것

  1. Html은 Html 코딩의 가장 시작과 끝
  2. Head는 바로 밑에 타이틀과 사용되어지는 형식
  3. Body는 코딩이 들어가는 본문

  4. Title= 홈피의 상단에 나오는 이름 설정

  5. H= HEADING

  6. a= 다른 페이지로 링크를 거는 기능

    7. a 뒤에 href(하이퍼 레퍼런스의 줄임말)="웹페이지 주소입력" 으로 설정함.
    링크 오픈시 새로운 창에서 열리길 원할 때에는 하이퍼 링크 주소 뒤에 target="_blank" 를 넣어준다.
    링크 위에 커서가 올려질 때 웹소개 글씨를 만들려면... title="이름" 추가하면 됨.

  7. U= 글 밑에 밑줄치는 기능

  8. IMG= 사진 추가 기능
